   Ferramenta para realização de ataques DDos (Dos) em redes wi-fi ou em seus hosts
 - Web site pessoal: https://linux4developers.blogspot.com
 - Repositorio: http://github.com/JumpAMS/DieHere

Como instalar:

Para instalar esta ferramenta primeiro precisaremos dar permisão de execução para a ferramenta de instalação
com o comando:

~$ sudo chmod +x Instalador.sh

em seguida já podemos instalar a ferramenta com o comando:

~$ sudo ./Instalador.sh

Como usar:
O objetivo desta ferramenta é ser extremamente facil de ser utilizada e muito simples.
Para utilizar a ferramenta basta digitar em seu terminal o comando:

~$ diehere

Então basta seguir as instruções.
